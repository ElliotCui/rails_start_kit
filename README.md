# RailsStartKit

Start kit for admin using rails.

```sh
  # clone the project
  $> git clone git@github.com:ElliotCui/rails_start_kit.git
  # change the the folder name(rails_start_kit) to YOUR_PROJECT_NAME as you wish
  $> mv(using `cp` is ok) rails_start_kit YOUR_PROJECT_NAME
  # enter the dictionary of YOUR PROJECT
  $> cd YOUR_PROJECT_NAME
  # init project with YOUR_PROJECT_NAME
  $> rails init:rename[YOUR_PROJECT_NAME]
  ...
  $> rake db:create
  $> rake db:migrate
  $> rake db:seed
  $> rails s
```
